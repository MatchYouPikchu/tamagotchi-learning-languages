Tamagotchi Pet Designer — Visual Catalog

Complete reference of all appearance options available in the pet designer.
Users can customize pets via preset theme chips (offline) or free text descriptions (LLM-powered, requires API key).

How Users Design Pets

MethodHowRequires
Theme ChipsClick one of 6 preset buttons (Magical, Sporty, Cool, Cute, Silly, Royal)Nothing — always available
LLM TextType a description like "a fluffy pink princess cat" and press EnterANTHROPIC_API_KEY env var
IterativeAfter LLM generates, type follow-up ("now add a hat") — conversation buildsANTHROPIC_API_KEY
ResetClick Reset to clear all customization back to defaultsNothing

Users do NOT pick individual options from dropdowns — they either use presets or describe in natural language. The LLM maps descriptions to the schema below.
