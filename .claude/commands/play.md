Launch the tamagotchi game for interactive testing:

```bash
cd ~/Documents/GitHub/tamagotchi && python3 main.py
```

The game window will open at 720x960 (480x640 at 1.5x scale). Key controls:
- **1-4**: Feed, Play, Clean, Sleep
- **ESC**: Back/Save & quit
- In Play menu: **1-6** to select game or Word Book

Tell the user the game is running and remind them of the key controls.
